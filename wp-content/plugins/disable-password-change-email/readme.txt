== Description ==
= Disable Password Change Email for WordPress =
Turns off the automatic "Password Lost/Changed" email messages sent to the admin when a user resets password.

= Usage =
* Nothing to configure. Simply activate and the emails stop coming!

== Changelog ==
= 1.0.0 =
* Initial release


== Installation ==
1. Download the *Disable Password Change Email plugin .zip file.
1. Unzip the archive
1. Upload the `disable-password-change-email` folder to `wp-content/plugins/`.
1. Go to the *Plugins* tab.
1. Activate *Disable Password Change Email*.
